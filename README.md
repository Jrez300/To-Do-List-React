## To-Do-List-App

This app is for those who need a reminder on what needs to be done. There are no limitations and can add as many task as you need 

## Getting Started
Simply Download repo or clone to local machine. Open code in your editor of choice and run ```npm i``` to download NPM module. Once NPM installed run ```npm start``` to launch the app in your browser. 

## Terminate app
To close the server and the react appliction , you can either type ```command + z``` on the command or just exit the browser.
